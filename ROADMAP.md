# NVSentinel Roadmap

This document outlines NVSentinel’s planned evolution from its current **Experimental** phase toward **Alpha**, **Beta**, and eventual **Production Stable**.

This roadmap will continue to evolve as we develop the project and receive feedback. This is a strategic list of objectives and will be augmented by our prioritization product board [here](https://github.com/orgs/NVIDIA/projects/133/views/1?template_dialog_tab=all&layout_template=board).

Any timelines are indicative and subject to change as we gather feedback and community contributions.

## Experimental
✅  Public release of NVSentinel repository  
✅  GPU Health Monitor + Fault Remediator prototype  
✅  Helm-based install and dry-run support

## Alpha
- [ ] CI and testing framework
- [ ] Contributor workflow and issue templates
- [ ] Complete user documentation & examples for major workflows  
- [ ] Improved observability integrations
  - [ ] Automated log collection
  - [ ] Support for cloud maintainence events (AWS, Azure, GCP, etc)
- [ ] Data store provider framework (pluggable data stores)
- [ ] Defined API and configuration schema
- [ ] GPU Operator integration testing

## Beta
- [ ] Scale and performance validation
- [ ] Secure release process

## Production Stable
- [ ] Production readiness
